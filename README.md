# vagrant-config-files
For this to work properly you need both the vagrant-vmware-desktop extension and the vagrant-vmware-utility installed.
To install the vagrant-vmware-desktop extension just run "vagrant plugin install vagrant-vmware-desktop" on your terminal.
To install the vagrant-vmware-utility go to https://developer.hashicorp.com/vagrant/docs/providers/vmware/vagrant-vmware-utility and follow the steps.
